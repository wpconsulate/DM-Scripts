DataTables Editor - extended form fields example

It can often be useful to have more information available in your form that you show in your table (the table for example would show a summary, while the form, with more screen real estate available to it can show detailed information). This is readily done with DataTables and Editor through the fact that DataTables can use objects as a data source. These objects provide all the information needed for the form, but only a limited selection of the available properties are shown in the table, while the form makes full use of the whole object.

This example (which is of course grossly simplified in order to make the point!) shows three fields only in the table, while the form provides editing abilities for seven different fields.
